# 开发计划

ZXMange 的目标是成为开发者桌面上最得心应手的全能工具箱。我们不仅仅满足于基础的项目管理，更希望通过深入集成开发工作流，提升每一位开发者的效率。

以下是我们的长期规划路线图，具体发布时间可能会根据开发进度进行调整。


## 🚀 近期计划 (v1.1.x) <Badge type="tip" text="开发中" />

我们正在努力完善核心功能的深度体验。

### 内置终端集成
- **功能描述**：无需切换到外部 CMD 或 Terminal，在 ZXMange 内部直接打开集成终端。
- **价值**：实现“一个窗口搞定所有操作”，支持执行 `npm install`、`git command` 等任意指令。
- **进度**：底层架构设计中...

### 环境管理增强
- **Node 版本切换**：可视化切换本地 Node 版本（需配合 nvm）。
- **依赖分析**：可视化查看 `package.json` 依赖树，检测过期或存在安全风险的依赖包。


## 🌟 中期规划 (v1.2.x - v1.6.x)

探索更多项目类型的深度支持与团队协作功能。

### 小程序开发深度支持
- **功能描述**：针对微信小程序项目，集成预览二维码生成、上传代码至微信后台等功能。
- **扩展**：支持 Uni-app 多端编译预览面板。

### 数据库管理模块
- **功能描述**：支持连接 MySQL、MongoDB、Redis 等常用数据库。
- **价值**：在管理代码的同时，直接查看和修改数据，实现真正的全栈开发闭环。

### 多项目管理视图
- **功能描述**：支持看板视图，通过拖拽卡片来管理多个项目的进度状态。
- **笔记云同步**：提供可选的云端服务，实现项目笔记在多台设备间同步。


## 🏗️ 远期愿景 (v2.0.0+)

构建 ZXMange 插件生态，由社区驱动无限可能。

### 插件系统
- **功能描述**：开放插件 API，允许第三方开发者编写插件扩展软件功能。
- **场景**：例如自动生成 CRUD 代码插件、UI 设计稿转代码插件等。
- **自定义主题**：支持用户自定义软件界面的颜色主题和布局。

### 团队协作版
- **功能描述**：支持团队创建，共享项目配置、API 文档和部署流程。
- **远程开发**：探索 SSH 远程连接服务器项目进行管理的可能性。
