# 快速开始

ZXMange 是一款基于 Electron+React+Vite+TS 开发的桌面端项目管理工具，旨在为开发者提供一站式的多类型项目管理解决方案。

## 安装与启动

### 环境要求
- Node.js（可选，用于部分项目功能）
- GIT（可选，用于本地部署功能）

### 安装步骤
1. 下载安装包
2. 运行安装程序，按照提示完成安装
3. 启动 ZXMange 客户端

### 首次使用
1. 启动软件后，进入主界面
2. 可选择「新建项目」或「导入项目」开始使用
3. 建好项目后请先将需要用的各类配置配置完成

## 主界面介绍
- **左侧导航栏**：项目列表、功能菜单
- **中间区域**：当前选中项目的内容展示
- **右侧面板**：项目配置、笔记等辅助功能
- **顶部工具栏**：常用操作快捷按钮
