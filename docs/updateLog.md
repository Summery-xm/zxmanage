# 更新日志
ZXMange 一直处于积极迭代中，致力于为开发者提供更高效的桌面端项目管理体验。以下是版本更新的详细记录。

## v1.0.0 <Badge type="tip" text="Latest" />

**发布日期: 2026-03-20**

### 🎉 初始发布
ZXMange 正式发布，基于 Electron + React + Vite + TS 技术栈构建。

### 🌟 核心功能
- **多语言项目管理**：全面支持 Vue、React、Uni-app、Java、微信小程序、PHP、Node.js、Cocos 等多种技术栈项目的统一管理。
- **项目导入**：支持导入本地已有项目，自动识别项目类型。
- **文件管理**：提供基础的文件浏览与管理功能。
- **代码管理**：支持简单的 Git 界面化管理，可视化查看文件状态。
- **桌面端体验**：原生桌面应用体验，响应迅速，操作便捷。
